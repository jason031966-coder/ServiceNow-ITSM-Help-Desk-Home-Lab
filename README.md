# ServiceNow-ITSM-Help-Desk-Home-Lab
# Overview
This project is a self-directed ServiceNow ITSM home lab built using a ServiceNow Personal Developer Instance. The purpose of this lab is to practice common IT support workflows that are used in help desk, Tier 1 support, and IT support technician roles.

The lab simulates real workplace support tasks such as creating incidents, resolving user issues, fulfilling service requests, documenting change requests, creating users, and writing knowledge base articles. Each section includes screenshots to show the steps completed inside ServiceNow.
